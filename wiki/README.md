## 阅读建议

### 对于新成员

1. **第一天**: 
   - 阅读 [入职文档](./入职文档.md) 的"项目概述"和"开发环境搭建"部分
   - 如果是 Java 开发者，阅读 [Python 快速入门](./Python快速入门-面向Java开发者.md) 的前几章
2. **第一周**: 
   - 完成开发环境搭建
   - 运行示例代码
   - 阅读核心代码文件
   - 查看测试代码
   - 掌握 Python 基础语法（参考 Python 快速入门）
3. **第一个月**:
   - 深入阅读 [架构设计](./架构设计.md)
   - 掌握异步编程和类型系统
   - 尝试小改动
   - 参与代码审查
   - 使用 [快速参考](./快速参考.md) 作为日常开发参考

### 对于有经验的开发者

- 快速浏览 [入职文档](./入职文档.md) 了解项目概况
- 重点阅读 [架构设计](./架构设计.md) 理解系统设计
- 使用 [快速参考](./快速参考.md) 作为开发时的速查手册
